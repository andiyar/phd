# Paper 4 Codebook

Framework analysis codebook — DREAMS Trial qualitative study.
Top-level codes are fixed (deductive). Sub-codes emerge inductively from data.

Last updated: 2026-06-22

---

## Communication
*Quotes about the patient's communicative capacity — ability to receive or produce communication, including verbal, non-verbal, and gestural acts.*

| Sub-code | Definition | Notes |
|---|---|---|
| Verbal | Patient conscious and communicating freely — answering questions, conversing | Approx RASS-PAL 0 to -1 |
| Limited verbal | Patient drowsy or reduced but producing words or short answers | Approx RASS-PAL -1 to -2 |
| Responsive to voice | Patient rousable or responsive to verbal stimulation | Approx RASS-PAL -2 to -3 |
| Responsive to touch | Patient responds minimally to physical stimulation but not to voice | Approx RASS-PAL -3 to -4 |
| Minimally responsive | Patient intermittently waking or showing minimal spontaneous awareness | Approx RASS-PAL -4 |
| Non-responsive | Patient unresponsive to voice and touch | Approx RASS-PAL -5 |
| Communicating needs | Patient verbally or non-verbally communicating a need or request | |
| Difficulty communicating | Patient unable to verbalise but retaining some non-verbal capacity (nods, gestures, pointing) | |
| Non-verbal | Documentation of non-verbal behaviours — facial expressions, body movements — without clear communicative intent | |
| Confused | Patient alert but cognitively vague, limiting meaningful interaction — dual code with SC (Delirium) when content warrants | |
| Change in responsiveness | Documentation of a shift in consciousness level within or between assessments | |

## Symptom Control
*Quotes about symptoms, their severity, management, and response to intervention.*

| Sub-code | Definition | Notes |
|---|---|---|
| Agitation | Documentation of agitation episodes, attributed to discomfort or distress | |
| Anxiety | Documentation of anxiety or anxious behaviours, distinct from agitation | |
| Breakthrough medication | Documentation of PRN/rescue medication use | |
| Breathing distress | Documentation of dyspnoea, SOB, or respiratory distress | |
| Circulatory failure | Documentation of circulatory deterioration — weak/absent pulse, pallor, mottling | |
| Cognition intact | Documentation of intact orientation and cognition — absence of delirium or confusion | |
| Comfort achieved | Overall impression of patient comfort or settled state | |
| Death rattle | Documentation of moist/noisy breathing or secretions (terminal respiratory secretions) | |
| Delirium | Documentation of confusion, disorientation, or hallucinations | |
| Dizziness | Documentation of dizziness or vertigo | |
| Drug effect | Documentation of a patient's positive response to a medication | |
| Fatigue | Documentation of increased drowsiness, sleeping more, or reduced alertness | |
| Myoclonus | Documentation of myoclonic jerking or involuntary movement | |
| Nausea | Documentation of nausea or vomiting | |
| Oedema | Documentation of peripheral oedema, swelling, or fluid leakage noted on examination or clinical description; inclusive of pulmonary oedema where relevant | |
| Pain | Documentation of pain complaint or pain behaviour | |
| Pain controlled | Documentation of pain absence or effective pain management | |
| Psychological distress | Documentation of emotional distress — tears, sadness, fear | |
| Swallowing difficulty | Documentation of dysphagia or difficulty swallowing, including medication administration impact | |
| Terminal phase breathing | Irregular agonal breathing pattern and/or apnoea at end of life (not really Cheyne-Stokes but sometimes called that) | |
| Thermal discomfort | Documentation of patient feeling hot, cold, or temperature-related discomfort | |

## Family
*Quotes about the family's presence, behaviour, emotional state, and role in care.*

| Sub-code | Definition | Notes |
|---|---|---|
| Family present | Family member present at bedside, overnight or during a shift | |
| No family present | No family visitors documented during the shift | |
| Family providing care | Family member actively participating in comfort measures | |
| Family concern | Family member expressing concern or worry about the patient's condition, without overt distress | |
| Family distress witnessed | Staff or family documenting family member's distress or upset | |
| Family satisfaction with care | Family expressing happiness or satisfaction with the care being provided | |
| Family supported | Staff providing support or reassurance to family | |
| Family discussion | Staff updating or discussing the patient's condition or prognosis with family | |

## Patient-Centred Care
*Quotes about care that respects the patient as a person — dignity, comfort, preferences, holistic care.*

| Sub-code | Definition | Notes |
|---|---|---|
| Comfort - Apparent | Observed impression of patient comfort framed in terms of the person's experience — clinician or family observing comfort | |
| Comfort - Self Report | Patient's own verbal report of comfort, wellbeing, or absence of distress | |
| Emotional wellbeing | Patient experiencing positive emotion or joy, connected to those around them | |
| Reassurance | Staff providing reassurance to a distressed patient | |
| Environment | Documentation of environmental changes to support comfort or dignity (e.g. single room) | |
| Physical comfort | Documentation of hands-on comfort interventions — repositioning, pressure relief, physical adjustments to ease discomfort | |
